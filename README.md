# Kulturo

Journal culturel personnel — suivez vos jeux, films et livres en un seul endroit.

## Stack

- **Frontend** — HTML / CSS / JavaScript vanilla (SPA)
- **Backend** — Supabase (Auth + PostgreSQL + RLS)
- **APIs** — TMDb · IGDB (via proxy Edge Function) · OpenLibrary · Groq (IA)
- **Hébergement** — GitHub Pages

## Fonctionnalités

- Bibliothèque avec filtres, tri, vue grille/liste
- Recherche automatique via APIs (jeux, films, livres)
- Recommandations IA personnalisées (Groq)
- Statistiques & journal des médias terminés
- Fil d'activité partagé entre utilisateurs
- PWA installable (iOS / Android)
- Thème dark / light

## Config

Copier `config.example.js` → `config.js` et renseigner les clés API.

## Démo

Accessible sans compte via le mode démo sur la page de connexion.
